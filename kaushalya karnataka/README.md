# kaushalya karnataka
Kaushalya-Karnataka is a hyper-local Android application designed to support self-employed skilled workers such as electricians, plumbers, carpenters, painters, and technicians. The application acts as a digital “Skill Showcase” platform where workers can create service portfolios, upload completed work photos, publish service pricing, and receive community feedback. The platform aims to create a trusted local ecosystem where residents can easily discover verified skilled laborers within their area. Generative AI (GenAI) will be used to improve profile descriptions, service recommendations, and customer engagement.
